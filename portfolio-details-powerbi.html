Project information

  • Category: Power BI
  • Client: PT. Jejaring Tiga Artha (Zicare)
  • Project date:2023 - 2024

Comprehensive Data Insights and Visualization for Business and Healthcare

Developed and implemented interactive dashboards for hospital management, tracking key metrics such as patient admissions, resource utilization, and financial performance. Enhanced data analysis for e-commerce sales trends by visualizing category-wise revenue and annual growth. Delivered insightful dashboards for employee performance evaluation, integrating multiple data sources to provide a holistic view for decision-making.

Features

  • Three Interactive Report Pages:
    • Finance Dashboard
    • Profitability Analysis
    • Time-Based Analysis
  • Dynamic Visualizations:
    • KPIs
    • Line Charts
    • Waterfall Charts
    • Treemaps
    • Decomposition Trees
  • DAX Calculations:
    • Year-to-Date (YTD) Sales
    • Previous Year Sales
    • Sales Growth Percentage

Dataset

Key Columns:

  • Segment
  • Country
  • Product
  • Sales
  • Profit
  • Discounts
  • Units Sold

Date Columns:

  • Date
  • Month
  • Year

Purpose

  • Provide a comprehensive overview of financial performance.
  • Identify top-performing products, segments, and regions.
  • Enable data-driven decision-making through visually appealing insights.

Key Visualizations

Report 1: Finance Dashboard

  • Card Visuals: Total Sales ($127.93M) dan Total Profit ($16.89M)
  • Waterfall Chart: Profit contribution by business segments
  • Matrix Visual: Sales dan profit margins by country dan year

Report 2: Profitability Analysis

  • Scatter Chart: Correlation between sales, profit, dan units sold
  • Stacked Column Chart: Segment-wise contributions to total sales dan profit

Report 3: Time-Based Analysis

  • Line Chart: Sales dan profit trends over time
  • Decomposition Tree: Break down profitability by country, segment, dan product

Setup Instructions

  1. Install Power BI Desktop.
  2. Familiarity dengan Power BI features dan DAX formulas.
  3. Clone repository: git clone https://github.com/jicsjitu/Financial-Dashboard.git
  4. Open file .pbix di Power BI Desktop.
  5. Interact dengan dashboard menggunakan slicers dan visuals.
